Project Blueprint: Weekly Budget & Monthly Savings Tracker
1. Project idea

Build a full-stack budgeting web app where a user can:

set a monthly income
set a weekly spending limit
add daily expense transactions
see which weeks were:
over budget
under budget
see monthly savings at the end of the month
Main goal

Help a user answer:

How much did I spend this week?
Did I go over my weekly budget?
How much did I save this month?
2. Why this project is good

This project is good for learning and for a resume because it shows:

TypeScript
frontend development
backend development
database design
API creation
business logic
validation
dashboard UI
deployment

It is focused, realistic, and not too huge.

3. Suggested stack
Frontend
Next.js
TypeScript
Tailwind CSS
React Hook Form
Zod
Backend
Next.js Route Handlers / API routes
TypeScript
Prisma ORM
Database
PostgreSQL
Deployment
Vercel
Neon or Supabase Postgres
4. MVP scope

Keep the first version simple.

Must-have features
user signup/login
create monthly budget
set monthly income
set weekly spending limit
add expense transactions
edit/delete transactions
weekly spending summary
monthly savings summary
